ERICO LOPES DE SOUZA - PROGRESS REPORT 08/03/2026

After many trials, I finally found the model for an airplane that had no holes on the skin and absolutely nothing modelled on the inside.

A smooth, watertight hull for the simulation. The model was found at Grabcad.com at the following link

https://grabcad.com/library/f35-1

After the high quality and suitable model was found, a .geo code was written to mesh the model and to re-scale the aircraft ( it was originally approximately 40m instead of the 15m).

Then, the VTK files were exported for 2 different levels of mesh ( 600K elements and 700K elements) but these numbers will change and more meshes will be done (4 total)

Now, no difficulties were found, I just need more time to finish the meshes.
